# Scripts
A short description of what each script does and where it is used.

## `occupations.py`
Used for theory paper.
Computes spin-flip occupations to check validity of spin-wave approx.
Compare different systems sizes, frustrated and critical branch, increasing temperature and single site occupations for comparison between OBC and PBC.

## `decayBranches.py`
Used for theory paper.
Compute decay rates.

## 'vertices.py'
Used for theory paper.
Explore vertices.
